Figure 1: How the Agent Learned RL Scoring Strategies
14 experiments over ~10 days on a real generative chemistry campaign
Run 1: Naive soft scoring
~25 soft Pareto objectives, no hard constraints. MW and rotatable bonds drift — Pareto dilution lets bad properties survive on other axes.
LESSON 1 Soft scoring alone fails under Pareto dilution
Runs 3–4: Invent HARD GATES
MW, RotBonds, AromaticRings as binary kill-switches. Out-of-range = molecule invalidated (not just penalized). Properties stay in bounds.
LESSON 2 Hard gates enforce property bounds. Implementation is fragile.
Run 5: Pharmacophore gate added
Donor must be 6–9 bonds from core. Pharmacophore enforced — but aromatic R-groups collapse to ~0% (unnoticed for several runs).
Run 7: Soft scoring push
Aggressive soft lipophilicity scoring (weight = 5.0×). FAILED: even at 5× weight, absorbed by ~29-objective Pareto dilution. Median unmoved.
LESSON 3 Soft scoring CANNOT overcome Pareto dilution at ~29 objectives
Run 8: Hard gate + cold start
Hard lipophilicity gate + COLD START (untrained prior). FAILED: 1–4/128 molecules survive the 5-gate stack. Agent starved.
LESSON 4 Cold start on tight gate stack starves the agent
Run 8b: Hard gate + warm start ✓
Same hard gate + WARM START from Run 7 checkpoint. SUCCESS: +1.85 unit lipophilicity shift. 42→86/128 survivors in 5 iterations.
LESSON 5 Warm-start + one new gate = the working recipe
Run 9: Composable gates ★ — Campaign Deliverable
Add metabolic stability gate, warm from 8b. SUCCESS: 5 simultaneous hard gates held. All properties in target windows.
LESSON 6 Hard gates are composable with proper warm-start chains
Runs 10–10e: Five attempts to reduce MW
v10: ceiling 580 → COLLAPSED   v10b: ceiling 600 → COLLAPSED   v10c: ceiling 640 → unchanged   v10d: anneal → 100% killed   v10e: weight=20 ("nuclear option") → MW 625.7, IDENTICAL to Run 9
LESSON 7 Some properties are TOPOLOGY-LOCKED (not free parameters)
Run 13: Shape–MW correlation test
Test if 3D shape scoring drives MW up (weight 5.0). MW WORSE (687). Shape creates MW↔volume correlation — reweighting can't break it.
LESSON 8 Correlated objectives can't be down-weighted — remove the source
Run 14: Remove correlation source
Remove shape-volume, keep size-normalized shape. MW only −13 Da, but donor coverage HALVED (49%→31%). The volume component was accidentally helping donors via chain extension.
LESSON 9 Scoring components have unintended side effects
ARC CLOSED
Run 9 remains the deliverable. Further improvement needs scaffold redesign, not scoring adjustment.

Figure 2: Where the PI's Input Changed the Course
Same campaign — only the moments where human expertise was essential
AGENT WORK
PI INTERVENTIONS
Runs 1–5: Building the gate stack
Agents iterating on scoring, learning hard-gate recipe. Properties converging. Everything looks fine in the data…
"Just 1 compound below MW 650. Why?"
PI noticed the problem from visual compound review in a 3D viewer. Agents had all the data but hadn't flagged it as anomalous.
Dim 1: Diagnosis
Runs 7–8: Designing A/B experiment
Agent designs controlled test of lipophilicity gate hypothesis. WARM-STARTS from biased checkpoint.
"Use cold start — you need a clean comparison."
PI caught experimental design error: testing whether a bias exists by starting FROM the biased state.
Dim 2: Exp. design
Runs 10–10e: Five MW reduction attempts
All fail identically. MW stuck at ~625. Agent proposes: "What if weight=20?" Then: "What about annealing?" Then: "Maybe a different checkpoint?"
"This is topology-locked. Stop trying."
PI recognized what 5 failed experiments meant: NOT "we haven't found the right setting" but "no setting exists." Agents would have run 5 more.
Dim 3+7: Stop / Reformulate
Curation: Maximum-diversity showcase
Agent builds 250-compound showcase using algorithmic diversity (MaxMin on fingerprints). 95.6% unique scaffolds. Looks great!
"I want SAR stories, not maximum spread."
Round 1 rejected. PI wanted thematic groupings that tell a med-chem narrative: "why is this compound interesting?" not "it's far from the others."
Dim 5: Aesthetic judgment
Curation Round 2: Themed selection
Agent rebuilds with 13 themes, iterates 3 rounds with critic loop. 202 compounds across SAR stories. Shipped to 3D viewer for review.
"R1 must have linker + exactly ONE ring."
Structural SAR rule from domain expertise. Not derivable from the data the agents had. PI's mental model of the binding site = ground truth.
Dim 6: Feature-level
Classification: Labeling by vector
Agent labels compounds by which structural vector was modified, using run/file names as proxy. 48 "triple-modified" compounds found!
"Those labels are WRONG. Decompose the structure, don't parse filenames."
PI caught systematic classification error. 48 "triples" were actually 34 single + 10 double + 0 triple. Propagated through entire database.
Dim 4: Shared state

THE BOTTOM LINE

AGENTS ran 14 RL experiments, predicted ADME on >60,000 compounds, docked thousands, curated selections, and maintained a lab notebook.
THE PI made ~6 critical interventions that each changed the trajectory. Without them: still running MW experiments on a topology-locked scaffold, with wrong labels, showcasing diverse but chemically uninteresting sets.
Agents did 95% of the work
PI made 95% of the decisions that mattered
Benchmark dimensions: 1 — Diagnosis under Pareto complexity 2 — Experimental design with controls 3 — Knowing when to stop 4 — Multi-agent shared state 5 — Aesthetic / qualitative judgment 6 — Feature-level generalization 7 — Reformulation vs. perseverance
